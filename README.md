# Arch-ReconKit

A fast and simple reconnaissance tool for Arch Linux systems. Designed for system administrators, digital forensics, and authorized security operations.

## Features

- System information collection (kernel, OS, hardware, storage, memory)
- User enumeration (users, groups, login history)
- Network configuration and open ports
- Organized timestamped reports
- Lightweight and dependency-free

## Usage

```bash
chmod +x reconkit.sh
./reconkit.sh
